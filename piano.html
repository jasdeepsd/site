Piano Song Resystheziation Software

Project Description

Transcribe a piano composition and re-systhesize the compostion into a mp3 file. Song chosen for this was Kiss the Rain by Yiruma.

Tasks and Skills

  • Created a script in Matlab to read in the mp3 file and applied fast Fourier theorem to find frequency. The frequencies were converted into notes.
  • The results of the code can be found in the download section. The first file is the original song and the second one is the song re-systhesized by the code.
